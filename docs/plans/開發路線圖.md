## 待辦項目

### 高優先順序

#### 1. DC 整合（Discord Integration）
- **描述**：將 MemoriaCore 接入 Discord 平台，作為另一個聊天渠道
- **目標**：讓使用者可透過 Discord 機器人與 AI 互動
- **相關檔案**：`api/telegram_bot.py`（可作為參考範例）
- **需求組件**：
  - [ ] Discord Bot 帳號設定（Application + Bot Token）
  - [ ] `api/routers/discord.py` — Discord 事件處理路由
  - [ ] `api/discord_bot.py` — Discord Bot 整合（類似 `telegram_bot.py`）
  - [ ] WebSocket 連線管理整合 Discord 事件
  - [ ] 頻道映射（`resolve_context` 支援 `channel='discord'`）
- **預估工時**：1-2 週

#### 2. AI 生圖功能（Image Generation）
- **描述**：支援 AI 圖像生成，可應用於角色視覺化、場景生成等
- **目標**：透過 LLM Provider 生成圖像並儲存至記憶系統
- **需求組件**：
  - [ ] Image Generation Tool 實作（`tools/image_gen.py`）
  - [ ] 支援 providers：DALL-E / Stable Diffusion / Flux 等
  - [ ] `core/persona_evolution/` — 人格視覺化輸出
  - [ ] 圖像儲存至 `StorageManager`（Blob 儲存）
  - [ ] 對話中觸發生圖的 router 規則（`DIRECT_CHAT_SCHEMA`）
  - [ ] 前端生圖 UI（dashboard.html 嵌入）
- **預估工時**：2-3 週
