## 待辦項目

### 高優先順序

#### 1. AI 生圖功能（Image Generation）
- **描述**：支援 AI 圖像生成，可應用於角色視覺化、場景生成等
- **目標**：透過 LLM Provider 生成圖像並儲存至記憶系統
- **需求組件**：
  - [x] Image Generation Tool 實作（`tools/minimax_image.py`）
  - [ ] 支援 providers：DALL-E / Stable Diffusion / Flux 等
  - [ ] 主動發送圖片機制：AI 偶爾會根據心情或對話內容，主動生成並傳送圖片給使用者（限定 SU 權限使用）。
- **預估工時**：2-3 週
